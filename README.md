# Dest Proposal (Mockup)

A simple design proposal for Dest - A free ride-share landing page concept for San Antonio, featuring an ad-sponsored electric shuttle service connecting riders with local businesses:
- Strong headline + short service description
- Section explicitly mentioning **contractors** and **real estate agents**
- Clear CTAs (“Book a Call” / “Get Started”)
- Interactive lead form (validation + success toast + copy-to-clipboard)
- No pricing
